## 🎓 EduHigh

EduHigh is a modern, static educational website inspired by English learning platforms.
It is designed for high school students to explore English content through a clean, structured, and engaging interface.

EduHigh focuses on delivering a simple but effective learning experience using well-organized pages and a user-friendly design.

## ✨ Overview
EduHigh helps students improve their English skills step by step through well-organized lessons, interactive exercises, and a smooth user experience.
The goal is simple: make English easier to learn, and more engaging to practice.

## 🚀 Key Features
📚 Structured English lessons tailored for high school level
🧠 Interactive exercises for active learning
🎯 Student-focused learning experience
🎨 Clean, modern, and responsive UI design
⚡ Fast and lightweight performance

## 🎯 Purpose
EduHigh was created to support high school students in mastering English in a way that feels natural, practical, and motivating.
It combines education with modern web design to create a better learning experience.
